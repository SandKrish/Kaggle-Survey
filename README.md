# Kaggle Survey Analysis

This repository contains analysis of the **Kaggle Machine Learning and Data Science Survey**.

The project explores trends and insights from the survey, focusing on how the **machine learning and data science landscape changed between 2021 and 2022**.

## Contents

**what-changed-from-2021-to-2022-in-ml-and-ds.ipynb**
A notebook analyzing survey results and highlighting key changes in the ML and Data Science community.

## Tools Used

* Python
* Pandas
* NumPy
* Data Visualization Libraries
* Jupyter Notebook
